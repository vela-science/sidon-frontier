# Statement

A Sidon set in {0,1}^n is one in which all pairwise componentwise sums are distinct; OEIS A309370 records the maximum size of such a set. Verified search on this frontier improved the published lower bounds for a(8) through a(24), and those bounds are now approved comments on the live A309370 entry. Every bound carries an explicit witness set that the frozen verifier re-checks from scratch: `vela reproduce examples/sidon-sets`.

— William Blair, 2026-06-11
